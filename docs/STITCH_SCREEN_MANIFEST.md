## Stitch reconstruction strategy

Use Stitch for:
- core journey frames
- important overlays
- responsible-AI differentiators

Use screenshots only for:
- repetitive loading variants
- repetitive empty states
- repetitive error states
- low-priority operational-state gallery frames
